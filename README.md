## 🛠️ Installation & Usage

Clone the script and use the multi-command suite to handle setup, telemetry monitoring, or a complete system teardown:

```bash
# Clone the repository
git clone https://github.com/hanzsath/filen_backup.git
cd filen_backup
chmod +x filen-automator.sh

# 1. Run the deployment wizard
./filen-automator.sh setup

# 2. Query real-time engine health state
./filen-automator.sh status

# 3. Cleanly wipe out configuration footprints & active timers
./filen-automator.sh remove
```
